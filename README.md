设计人：刘星泉

本项目为港口BI（商业智能）原型系统，旨在通过数据可视化与实时监控，助力港口运营管理智能化升级。系统以“悉数（Insight）”为核心品牌，整合港口全链路关键数据，覆盖生产、安全、设备、能耗等多维度，为管理者提供直观、高效的决策支持。

核心功能与界面亮点

首页概览：以横向图片墙展示港口关键场景（公司领导、民航、港口实景等），搭配导航栏快速切换模块，直观呈现港口核心要素。

生产效率监控：

航线生产效率：展示航线总览（203条航线、总吞吐量/作业量）、类型占比饼图、TOP5航线吞吐量/作业量排名，辅以箱量趋势与效率趋势折线图。

设备生产效率：实时统计岸桥、龙门吊等设备作业时长（如全部设备940.5h）、单机对位/停时/告警数据，通过折线图呈现作业时长、净效率、利用率趋势，底部附设备实时状态表格（含设备号、状态、能耗等）。

闸口生产效率：深色背景下展示闸口缓冲时长、异常问题处理趋势，右侧统计进出闸量、箱量、识别率（86.3%）、直通率（79.5%），辅以放行次数比例、滞场时间占比圆形图表。

堆场生产效率：通过柱状图分类统计在场箱量（空箱、重箱、危险品箱等），仪表盘展示堆存率（86.3%）、空箱利用率（79.5%），折线图追踪平均堆存周期、翻箱率趋势。

实时数据可视化：

3D港口场景模型动态展示集装箱、龙门吊、卡车分布，标注泊位利用率（89%）；卫星地图叠加三角形标识，直观呈现港口布局。

吞吐量目标完成趋势图、航线/船公司吞吐量排名（如北大西洋航线4389万TEU居首）、内外贸/进出口占比饼图（外贸9.4%、内贸40.9%）。

智能监控与预警：

“智慧云控”模块集成12路监控画面（如A-36号监控），实时捕捉集装箱、起重机、船舶作业动态。

船舶作业进度跟踪（如“高丽大连”轮完成78.3%）、识别准确率（最高99.49%）、危险品箱量（404,540TEU）等关键指标预警。

辅助功能：涵盖安全管理（作业、堆场、船舶等场景图片库）、能耗管理、客户管理、财务管理，支持按分公司（如北二分公司）、时间维度筛选数据。

界面设计特色

深色主题+橙色高亮：数据图表色彩丰富（红/蓝/绿区分趋势），关键指标（如完成率、利用率）突出显示。

多区域协同布局：左侧功能导航栏（港口概况、生产管理等）、中部核心图表（折线图/饼图/仪表盘）、右侧排名/统计框，底部实时状态表格，层级清晰。

实时性与交互性：顶部显示当前时间（如11:20:37）、天气（42℃），支持超时报文（如4条待处理）、目标未达标（如月度完成率告警）等动态提醒。

本原型系统通过整合港口全要素数据，实现“可视、可析、可控”的智能管理，为提升作业效率、优化资源配置提供数据支撑。

English README Introduction

This project is a Port BI (Business Intelligence) Prototype System, designed to enhance port operation management through data visualization and real-time monitoring. Centered on the brand "Insight," it integrates key data across the entire port chain—including production, safety, equipment, and energy consumption—to provide intuitive decision support for managers.

Core Features & Interface Highlights

Homepage Overview: Displays key port scenarios (company leaders, civil aviation, port views) via a horizontal image wall, with a navigation bar for quick module switching, offering a direct overview of core port elements.

Production Efficiency Monitoring:

Route Production Efficiency: Presents route overviews (203 total routes, total throughput/operation volume), pie charts for route type proportions, TOP5 route throughput/operation volume rankings, and line charts for container volume/efficiency trends.

Equipment Production Efficiency: Tracks real-time operation durations (e.g., 940.5h for all equipment) of quay cranes, gantry cranes, etc., displays single-machine positioning/downtime/alarm data, and uses line charts for operation duration, net efficiency, and utilization trends. A bottom table shows real-time equipment status (equipment ID, status, energy consumption, etc.).

Gate Production Efficiency: On a dark background, shows gate buffer duration and abnormal issue handling trends. Right-side stats include vehicle/container entry-exit volumes, recognition rate (86.3%), straight-through rate (79.5%), and circular charts for release frequency and detention time proportion.

Yard Production Efficiency: Uses bar charts to classify in-yard container quantities (empty, heavy, hazardous, etc.), dashboards for storage rate (86.3%) and empty container utilization (79.5%), and line charts for average storage cycle/relocation rate trends.

Real-Time Data Visualization:

3D port models dynamically display container/gantry crane/truck distribution, with berth utilization (89%) labeled. Satellite maps with triangular markers visualize port layout.

Throughput target completion trend charts, route/shipping company throughput rankings (e.g., North Atlantic Route leading at 43.89M TEU), and pie charts for domestic/foreign trade proportions (9.4% foreign, 40.9% domestic).

Intelligent Monitoring & Alerts:

The "Smart Cloud Control" module integrates 12 monitoring feeds (e.g., A-36 Monitoring) to capture real-time container/crane/ship operations.

Tracks vessel operation progress (e.g., "Gaoli Dalian" at 78.3% completion), identification accuracy (up to 99.49%), and hazardous container volume (404,540 TEU) with alerts.

Auxiliary Functions: Covers safety management (image libraries for operations, yards, vessels), energy/ customer/financial management, and supports filtering by branch (e.g., North Second Branch) and time.

Interface Design Features

Dark Theme + Orange Accents: Richly colored charts (red/blue/green for trends) highlight key metrics (completion rate, utilization).

Multi-Zone Layout: Left navigation (port overview, production management), center core charts (line/pie/dashboard), right rankings/stats, and bottom real-time tables for clarity.

Real-Time Interactivity: Top displays current time (e.g., 11:20:37), weather (42°C), and dynamic alerts (e.g., 4 overdue documents, target non-compliance warnings).

This prototype integrates full-port data to enable "visible, analyzable, and controllable" intelligent management, supporting efficiency improvement and resource optimization.
